study repo for various transpilation and embedding approaches

- haxe: a object oriented (OO) language that can compile to multiple languages' source code.. (ml, haxe)

- hy: a lisp that transpiles to Python AST. (Python)
- borgo: a rust like dynamic language that transpiles to Go. (Rust, Go)

- squirrel: a tiny and dynamic OO language for embedding. (C++)
- wren: small dynamic OO language for embedding. (C)

- `misc/` has:
  - c2
  - c3
  - lua
  - nit
  - nature
  - berry
  - pocket
  - janet
  - gravity
